Runtime-provided database drivers
---------------------------------

The shaded JAR does not bundle MySQL Connector/J. If you use MySQL, provide the
driver on the runtime classpath yourself:

- com.mysql:mysql-connector-j:9.2.0 - GPLv2 with Universal FOSS Exception 1.0
- com.google.protobuf:protobuf-java:4.29.0 - BSD-3-Clause

See README.md for the runtime classpath example.
